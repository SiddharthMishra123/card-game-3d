# 3D Card Game - Rummy & Poker 🃏

Ek badiya web-based 3D card game jismein **Rummy** aur **Poker** dono hain! Three.js ke through stunning 3D graphics! 🎮

## Features

### Rummy 🃏
- Classic Indian Rummy gameplay
- 13-card deal mechanics
- Sets aur Sequences (Meld system)
- Real-time opponent AI
- 4 players support
- Score calculation

### Poker 🎰
- Texas Hold'em variant
- Betting system (All-in, Fold, Raise, Call)
- Hand rankings
- Pot management
- 6 players support
- Blinds system

### 3D Graphics ✨
- Realistic 3D cards
- Smooth animations
- Interactive 3D table
- Beautiful UI

## Tech Stack

- **Three.js** - 3D rendering
- **Node.js** - Backend
- **Express.js** - API
- **Socket.io** - Real-time multiplayer
- **HTML5/CSS3/JavaScript** - Frontend

## Installation

```bash
npm install
npm start
```

Browser mein: **http://localhost:3000**

## Deployment

**Render par live hai:** https://card-game-3d.onrender.com

## How to Play

### Rummy
1. 13 cards har player ko
2. Draw karo deck se ya discard pile se
3. Sets aur Sequences banao
4. First ko empty hand wins!

### Poker
1. 2 hole cards
2. Bet karo
3. Community cards reveal hote hain
4. Best hand jeetega!
